# Proto-Zero - SpaceShooter

My first ever Flutter project.

## Getting Started

Gameplay :
- Drag around the player character (red jet).
- Player character automatically shoots projectiles with interval.
- Player gets 3 lives total before game over.
- Enemies spawn from upper half of screen and bounce around.
- Enemies shot give points and disappear.
- Gain most points before eventually dying.
